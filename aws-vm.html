AWS & Virtual Machines

Understanding Amazon EC2 (Elastic Compute Cloud), the backbone of IaaS (Infrastructure as a Service), and how to provision virtual servers in the cloud.

Infrastructure as a Service

1. What is an EC2 Instance?

Think of an EC2 instance as a virtual computer running in Amazon's data center that you can rent by the second. Instead of buying physical hardware, you "provision" a virtual machine (VM) with specific CPU, memory, and storage configurations.

2. Anatomy of an Instance

When launching a VM, you must configure these four pillars:

AMI (Amazon Machine Image)

The software template. It contains the Operating System (e.g., Ubuntu 22.04, Amazon Linux 2, Windows Server) and pre-installed software.

Instance Type

Determines the hardware capacity (CPU & RAM). Naming convention: t2.micro

  • T-series (General): Good for web servers, dev environments.
  • C-series (Compute): High performance processors for batch processing.
  • M-series (Memory): High RAM for databases.
EBS (Elastic Block Store)

The persistent hard drive attached to the instance. The OS lives here. If you stop the instance, data on EBS persists (unlike instance store/RAM).

3. Security & Connectivity

Security Groups

Acts as a virtual firewall controlling traffic to and from the instance. By default, all inbound traffic is blocked.

# Example Rules:
Allow TCP 22 (SSH) from My_IP
Allow TCP 80 (HTTP) from 0.0.0.0/0
Allow TCP 443 (HTTPS) from 0.0.0.0/0

Key Pairs (.pem)

AWS does not use passwords for Linux instances by default. You use a cryptographic key pair.

  • Public Key: Stored by AWS on the VM.
  • Private Key: Downloaded by you (.pem file). You must keep this safe; if lost, you cannot access the instance.

4. Lifecycle & Pricing

Understanding the state of your machine affects your bill.

State Description Billing Status
Running Instance is booted and active. Charged per second/hour.
Stopped Instance is shut down (like turning off power). Data on EBS is saved. Not charged for compute (only for EBS storage).
Terminated Instance is permanently deleted. Cannot be recovered. Billing stops completely.
